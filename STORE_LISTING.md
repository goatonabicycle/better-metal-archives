# Web Store Listing

## Short Description (132 chars max)

Filter Metal Archives by genre, location, and status. Hover over any band for a quick preview.

## Full Description

How many death metal bands are in South Africa? Metal Archives doesn't make this easy to find out.

This extension adds filtering and search to Metal Archives country pages. Browse any country's bands and filter by genre, status, location, or search by name. Statistics show genre and location breakdowns at a glance.

Hover over any band link on the site to see a preview with their logo, photo, genre, status, label, and recent releases.

**Features**

- Filter bands by genre (Death, Black, Thrash, etc.)
- Filter by status (active, split-up, on hold)
- Filter by city or region
- Text search across all fields
- Statistics: genre counts, location breakdown, status totals
- Band hover preview on all pages

Works on country band lists and label pages.

Open source: https://github.com/rowanhen/better-metal-archives
