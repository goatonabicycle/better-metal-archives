# better-metal-archives

Encyclopedia Metallum is a vital website. It is a form of truth that the metal world really benefits from. Discovering bands and having a form of index are such awesome uses of this website.

But it feels like it comes from 1998.

This extension aims to add some useful features that hopefully makes your time on this website a tiny bit more pleasant.
